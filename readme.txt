== Description ==

[Live Demo](https://themepoints.com/servicebox) | [Support](https://themepoints.com/questions-answer/)

Service Box is yet another simple, responsive, lightweight plugin for creating responsive service box with various styles.


<strong>Plugin Features </strong>

* Fully responsive.
* Clean Design & Code.
* No Coding Required.
* 3 Different Service Style.
* Easy Service Option Page.
* Display Service Box by group.
* Easy Font Awesome Icon Picker.
* Cross-browser compatibility.
* Add/remove Service item.
* Unlimited color options.
* Multicolour Service Box.
* Service Title Font Color Option.
* Service Content Font Color Option.
* Service Icon Color Option.
* Service Background Color Option.
* Unlimited Service box anywhere in the themes or template.
* And much more!



== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Service Box" menu at left side on WordPress dashboard and find "Service Box Options"<br />
<br />

<strong>Using FTP</strong>
1.Extract service-boxs.zip to your computer.
2.Upload the service-boxs directory to your wp-content/plugins directory.
3.After activate plugin you will see "Service Box" menu at left side on WordPress dashboard and find "Service Box Options"

<br />
<strong>How to use short-code?</strong><br />
When Service Box options setup done please copy & paste shortcode into your theme any where you want to display this service box.
